# 📥 資料置き場（.inbox）

## このフォルダの使い方

準備した資料をここに置いてください。AIが自動的に読み取って、適切な場所に整理します。

### 入れられるもの
- 📝 テキストファイル（.txt, .md, .docx）
- 🖼️ 画像ファイル（.png, .jpg, .jpeg）
- 📊 設計書（.pdf, .xlsx）
- 💬 チャット履歴（.txt, .md）

### フォルダ構成
```
.inbox/
├── specs/       # 仕様書・要件定義
├── images/      # 画面デザイン・UI案
├── notes/       # アイデアメモ
└── references/  # 参考資料
```

### 使用例
1. 画面の手書きスケッチ → `images/`に
2. 機能一覧のメモ → `specs/`に
3. 参考サイトのURL一覧 → `references/`に

---
⚠️ **注意**: このフォルダの内容は、AIが処理した後に自動的に整理されます。
