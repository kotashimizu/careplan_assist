## 💡 技術スタック概要

### フロントエンド
- **Next.js 14** - Reactフレームワーク
- **TypeScript** - 型安全な開発
- **Tailwind CSS** - スタイリング

### バックエンド
- **Supabase** - 認証・データベース
- **PostgreSQL** - リレーショナルDB
- **Row Level Security** - セキュリティ

### インフラ
- **Vercel** - ホスティング
- **GitHub** - バージョン管理
- **Docker** - 開発環境

## ⚠️ 技術的な注意事項

- API利用時は必ず認証を実装
- データベース変更時はマイグレーション使用
- 本番環境では環境変数を適切に設定
