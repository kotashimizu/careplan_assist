## 🤖 AIへの効果的な質問方法

### 基本の質問テンプレート
```
「[ドキュメントのパス]を読みましたが、
[具体的な箇所]が理解できません。
初心者にも分かるように説明してください」
```

### よくある質問パターン

#### 1. 始め方が分からない
```
「プログラミング初心者です。
何から始めればいいか教えてください」
→ AIが「00-ようこそ」フォルダを案内します
```

#### 2. 機能の実装方法
```
「[機能名]を実装したいです。
参考になるドキュメントはありますか？」
→ AIが適切なドキュメントを提案します
```

#### 3. エラーの解決
```
「[エラーメッセージ]が出ています。
トラブルシューティングを見ても解決できません。
原因と解決方法を教えてください」
```

#### 4. 次のステップ
```
「[完了した内容]まで終わりました。
次は何をすればいいですか？」
→ AIが次のドキュメントや作業を提案します
```

### 💡 効果的な質問のコツ

1. **具体的に伝える**
   - NG: 「動きません」
   - OK: 「ログインボタンを押してもエラーが出ます」

2. **状況を説明する**
   - 何をしようとしたか
   - どんなエラーが出たか
   - 期待する動作は何か

3. **ドキュメントを参照する**
   - 「○○ドキュメントの△△の部分について」
   - 関連するファイルパスを伝える
